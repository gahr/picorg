# picorg - a simple picture organizer

```
Usage: ./picorg.tcl -dir <path> -fmt <format> [-verbose] [-help]

-dir <path>    Process files in <path>
-fmt <format>  Store processed files according to <format>
               %Y  year
               %m  month
               %d  day
               %H  hour
               %M  minute
               %S  second
               %f  file name
-verbose       Print debug information
-help          Print this help message and quit
```
